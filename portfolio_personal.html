Mi nombre es Felipe Huang, y trabajo en la industria del desarollo web hace 5 años.

Como Desarollador Web Full Stack soy capaz de crear aplicaciones web responsive desde cero, armando tanto el front-end como el back-end y para esto cuento con conocimientos en distintos lenguajes de programación, bases de datos, sistemas de control de versiones y prueba de software.

Algunos de mis proyectos:

Juego de la Serpiente

Un juego de la serpiente en Python, utilizando Pygame para la interfaz gráfica y algoritmos de detección de colisiones. Estructurado en módulos para facilitar la escalabilidad y mantenibilidad del proyecto.

Juego del TaTeTi

El proyecto de Tateti en Python es un juego interactivo con lógica de matrices y funciones. Su creador busca mejorar sus habilidades de programación y la capacidad para implementar algoritmos de lógica en Python.

Juego del Ahorcado

El proyecto de programación del juego del ahorcado en Python utiliza Pygame para la interfaz gráfica. La lógica del juego involucra la manipulación de cadenas y el control de flujos de ejecución en Python.
